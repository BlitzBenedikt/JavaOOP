Objektorientierte Programmierung (OOP) ist ein Programmierparadigma, das auf der Idee von Klassen und Objekten basiert. Es ermöglicht uns, Code wiederzuverwenden und Strukturen effizient zu organisieren.

Grundkonzepte:

  • Klassen: Baupläne für Objekte. Eine Klasse definiert Eigenschaften und Methoden.
  • Objekte: Instanzen von Klassen, die konkrete Daten enthalten.
  • Vererbung: Mechanismus, bei dem eine Klasse Eigenschaften und Methoden von einer anderen erbt.
  • Polymorphismus: Fähigkeit von Objekten, auf unterschiedliche Weise auf Methoden zu reagieren.
  • Abstraktion: Das Herausfiltern der wesentlichen Eigenschaften eines Objekts.
  • Kapselung: Die Sicherstellung, dass Daten nur über definierte Schnittstellen zugänglich sind.

Erklärungsblock: Klassen vs. Objekte

Was ist der Unterschied zwischen Klassen und Objekten?

Klassen: Eine Klasse ist wie eine Schablone oder ein Bauplan, der die Struktur und das Verhalten von Objekten definiert. Sie beschreibt die Eigenschaften (Attribute) und Funktionen (Methoden), die ein Objekt besitzen kann.

Objekte: Ein Objekt ist eine Instanz einer Klasse. Es wird basierend auf dem Bauplan der Klasse erstellt und kann mit eigenen Werten gefüllt werden. Während die Klasse eine allgemeine Vorlage ist, repräsentiert das Objekt eine spezifische Realisierung dieser Vorlage.

Beispiel: Wenn eine Klasse Auto existiert, definiert sie allgemeine Eigenschaften wie Marke und Baujahr. Ein Objekt dieser Klasse könnte ein spezifisches Auto wie "BMW, Baujahr 2023" darstellen.

Weitere Details: Klassen dienen als Schablone, in der Attribute und Methoden festgelegt werden. Jedes Objekt stellt eine konkrete Realisierung dieser Schablone dar, wobei es eigene Werte und Zustände besitzt. Wenn Sie mehrere Objekte aus derselben Klasse erstellen, teilen sie sich den Bauplan, unterscheiden sich aber durch ihre konkreten Eigenschaften (z.B. Name, Alter).

Aufgaben:

Aufgabe 1: Klasse erstellen

Erstellen Sie eine Klasse Person. Die Klasse sollte folgende Eigenschaften und Methoden enthalten:

  • Eigenschaften: Name (String), Alter (int)
  • Methode: vorstellen(), die den Namen und das Alter der Person in der Konsole ausgibt.

Hinweis: Denken Sie daran, dass die Methode vorstellen() mit System.out.println() arbeitet, um die Ausgabe zu erzeugen.

Beispielausgabe:

Mein Name ist Anna und ich bin 25 Jahre alt.
        
public class Person {
    String name;
    int alter;

    public void vorstellen() {
        System.out.println("Mein Name ist " + name + " und ich bin " + alter + " Jahre alt.");
    }
}

Aufgabe 2: Objekte erstellen

Erstellen Sie zwei Objekte der Klasse Person mit eigenen Werten. Verwenden Sie die Methode vorstellen(), um die Werte der Objekte in der Konsole auszugeben.

Schritte:

  1. Erstellen Sie ein Objekt der Klasse Person mit einem Namen und einem Alter Ihrer Wahl.
  2. Rufen Sie die Methode vorstellen() für dieses Objekt auf.
  3. Wiederholen Sie dies für ein zweites Objekt mit anderen Werten.

Beispielausgabe:

Mein Name ist Anna und ich bin 25 Jahre alt.
Mein Name ist Max und ich bin 30 Jahre alt.
        
public static void main(String[] args) {
    Person anna = new Person();
    anna.name = "Anna";
    anna.alter = 25;
    anna.vorstellen();

    Person max = new Person();
    max.name = "Max";
    max.alter = 30;
    max.vorstellen();
}

Erklärungsblock: Vererbung

Was ist Vererbung?

Vererbung ist ein Mechanismus, bei dem eine Klasse (die Basisklasse) ihre Eigenschaften und Methoden an eine andere Klasse (die abgeleitete Klasse) weitergibt. Dadurch können Programmierer bestehenden Code wiederverwenden und neue Funktionalitäten hinzufügen.

Beispiel:

class Fahrzeug {
    String typ;

    void starten() {
        System.out.println("Das Fahrzeug startet.");
    }
}

class Auto extends Fahrzeug {
    String marke;

    void hupen() {
        System.out.println("Das Auto hupt.");
    }
}

In diesem Beispiel erbt die Klasse Auto von Fahrzeug. Sie hat Zugriff auf die Methode starten() und fügt zusätzlich die Methode hupen() hinzu.

Aufgabe 3: Vererbung implementieren

Erstellen Sie eine Basisklasse Tier mit den folgenden Eigenschaften und Methoden:

  • Eigenschaften: Name (String), Alter (int)
  • Methode: essen(), die ausgibt: [Name] isst gerade.

Erstellen Sie eine abgeleitete Klasse Hund, die von der Klasse Tier erbt. Fügen Sie der Klasse Hund eine Methode bellen() hinzu, die ausgibt: [Name] bellt.

Schritte:

  1. Definieren Sie die Klasse Tier mit den oben genannten Eigenschaften und Methoden.
  2. Erstellen Sie die Klasse Hund, die von Tier erbt.
  3. Erstellen Sie ein Objekt der Klasse Hund, setzen Sie die Werte für Name und Alter, und rufen Sie die Methoden essen() und bellen() auf.

Beispielausgabe:

Bello isst gerade.
Bello bellt.
        
class Tier {
    String name;
    int alter;

    void essen() {
        System.out.println(name + " isst gerade.");
    }
}

class Hund extends Tier {
    void bellen() {
        System.out.println(name + " bellt.");
    }
}

public static void main(String[] args) {
    Hund bello = new Hund();
    bello.name = "Bello";
    bello.alter = 3;
    bello.essen();
    bello.bellen();
}
